# 第三章 一元一次方程

## 3.1 什么是方程

**方程**是含有未知数的等式。例如：`2x + 3 = 7` 就是一个方程，其中 `x` 是未知数。

方程的两个基本特征：
1. 必须是等式（含有等号 `=`）
2. 必须含有未知数（通常用字母表示）

### 练习：判断下列式子哪些是方程

```
(1) 3 + 5 = 8          → 不是方程（没有未知数）
(2) 2x + 1 = 5         → 是方程
(3) y - 3 > 7          → 不是方程（不是等式）
(4) 4a + 2 = 10        → 是方程
```

---

## 3.2 一元一次方程

**一元一次方程**是指含有一个未知数，且未知数的最高次数为 1 的方程。

- **一元**：一个未知数
- **一次**：未知数的次数是 1

标准形式：`ax + b = 0`（其中 `a ≠ 0`）

例如：`3x + 5 = 20` 是一元一次方程，`2x² + 3 = 11` 则不是（x 的次数是 2）。

---

## 3.3 等式的性质

### 性质 1
等式两边同时加上（或减去）同一个数（或式子），等式仍然成立。

如果 `a = b`，那么：
- `a + c = b + c`
- `a - c = b - c`

### 性质 2
等式两边同时乘以（或除以）同一个不为 0 的数，等式仍然成立。

如果 `a = b`，那么：
- `a × c = b × c`
- `a ÷ c = b ÷ c`（其中 `c ≠ 0`）

---

## 3.4 解一元一次方程的步骤

解一元一次方程的一般步骤（简称"一去二移三化四除以"）：

| 步骤 | 操作 | 示例：`2(x - 3) + 5 = 3x + 1` |
|------|------|-------------------------------|
| ① 去分母 | 方程两边乘以各分母的最小公倍数 | （本例无分母） |
| ② 去括号 | 用分配律去掉括号 | `2x - 6 + 5 = 3x + 1` |
| ③ 移项 | 含未知数的项移到左边，常数项移到右边 | `2x - 3x = 1 + 6 - 5` |
| ④ 合并同类项 | 分别计算左右两边 | `-x = 2` |
| ⑤ 系数化为 1 | 两边除以未知数的系数 | `x = -2` |

### 例题 1：基本的一元一次方程

解方程：`3x + 5 = 20`

**解：**
```
3x + 5 = 20
3x = 20 - 5     ← 移项（+5 移到右边变成 -5）
3x = 15         ← 合并同类项
x = 5           ← 系数化为 1（两边除以 3）
```

**验算：** 将 `x = 5` 代入原方程：
左边 = `3 × 5 + 5 = 15 + 5 = 20` = 右边 ✅

### 例题 2：含括号的方程

解方程：`2(x - 3) = 8`

**解：**
```
2(x - 3) = 8
2x - 6 = 8      ← 去括号
2x = 8 + 6      ← 移项
2x = 14         ← 合并同类项
x = 7           ← 系数化为 1
```

### 例题 3：移项易错点

解方程：`5x - 3 = 2x + 9`

**解：**
```
5x - 3 = 2x + 9
5x - 2x = 9 + 3   ← 移项要变号！（2x 移到左边变成 -2x，-3 移到右边变成 +3）
3x = 12
x = 4
```

> ⚠️ **移项易错提醒**：移项一定要变号！正变负，负变正。

## 3.5 列方程解应用题

### 解题步骤

1. **审题**：理解题意，找出已知量和未知量
2. **设未知数**：用字母表示未知量
3. **找等量关系**：分析题目中的等量关系
4. **列方程**：根据等量关系列出方程
5. **解方程**：求解方程
6. **检验作答**：检验结果是否符合题意，写出答案

### 例题 4：和差倍分问题

小明有弹珠，小红的弹珠是小明的 2 倍，两人一共有 36 颗。问小明和小红各有多少颗弹珠？

**解：**
设小明有 `x` 颗弹珠，则小红有 `2x` 颗。

```
x + 2x = 36
3x = 36
x = 12
```

小明有 12 颗，小红有 `2 × 12 = 24` 颗。

**答：** 小明有 12 颗弹珠，小红有 24 颗弹珠。

### 例题 5：行程问题

A、B 两地相距 240 千米，甲车从 A 地出发，乙车从 B 地出发，相向而行。甲车速度 60 km/h，乙车速度 40 km/h。问几小时后两车相遇？

**解：**
设 `x` 小时后两车相遇。

等量关系：甲车路程 + 乙车路程 = 总距离

```
60x + 40x = 240
100x = 240
x = 2.4
```

**答：** 2.4 小时后两车相遇。

---

## 本章小结

- **方程** = 含有未知数的等式
- **一元一次方程** = 一个未知数 + 次数为 1
- **解方程五步**：去分母 → 去括号 → 移项 → 合并同类项 → 系数化为 1
- **移项要变号**是最大的易错点
- **列方程解应用题六步**：审题 → 设未知数 → 找等量关系 → 列方程 → 解方程 → 检验作答
